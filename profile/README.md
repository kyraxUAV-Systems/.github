## Autonomous Small Fixed-Wing UAV for Disaster Response

This organization contains projects related to the development of an **autonomous small fixed-wing UAV** designed for **disaster response and search-and-rescue operations**.

The goal is to build a system capable of **mapping disaster areas, performing aerial reconnaissance, and detecting entrapped survivors**. The platform focuses on lightweight deployment and autonomous operation in environments where ground access is limited or infrastructure is damaged.

### Core Technologies

* SLAM-based navigation
* Autonomous flight control
* Sensor fusion
* Embedded systems
* Aerial reconnaissance
* Survivor detection

### Purpose

Rapid situational awareness is critical in disaster scenarios.
This project explores how **autonomous UAV systems** can assist first responders by providing fast mapping and reconnaissance of affected areas.
